This is an Agentic Chatbot Project.

For the initial setup. Follow the steps 

Step 1: Install the dependencies mentioned in requirements.txt 
Step 2: Create all the necessary API's
Step 3: Connect to the backend first using the URL (the one you added in your backend.py file)
Step 4: Run the frontend.py using (streamlit run frontend.py)

After this, it's your game to play with the agent. 

Happy Learning!!
